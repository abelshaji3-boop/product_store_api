# 🛍️ Product Store

A simple full-stack Product Store application built using React.js, Express.js, Axios, and Tailwind CSS.

## 🚀 Features

- Fetch products from DummyJSON API
- Express.js Backend API
- React Frontend
- Axios for API calls
- Responsive Product Cards
- Tailwind CSS Styling
- Component-Based Architecture

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- CORS
- Axios
